1. Linear Regression
* x and y has a correlation 
# Equation 
* m = slope ; b = y intercept
~ y = mx + b
~ (y2 - y1) / (x2 - x1) = m 

*RESIDUAL = difference between actual and predicted
~ r = actual - predicted
 
2. K Nearest Neighbor (KNN)
* k is a parameter
* eg. k = 5 (should be odd)
* find the 5 closest datapoint to the input
* classify the input based on the highest recuring datapoint
# Equation 
* m = magnitude, d = distance
~ d = sqrt[(x2-x1)^2 + (y2-y1)^2]
